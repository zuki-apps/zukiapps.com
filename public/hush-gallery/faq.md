## What is Hush Gallery?

Private photo and video gallery. Media stored in app-private storage, organized with tags, optional Premium App Lock (PIN/biometric), optional cloud backup (iCloud / Google Drive).

## Import & share

**Q: How do I import photos?**  
A: Tap + → Import from Gallery / Take Photo / Take Video. Optionally delete originals after import.

**Q: Share from WhatsApp or Photos?**  
A: Select media → Share → Hush Gallery. iOS uses Share Extension; Android uses share intent.

**Q: Delete originals after import?**  
A: Per-import dialog, or Settings → Deletion → Auto-Delete Source Files. iOS uses Photos asset IDs when available.

## Tags & organization

**Q: Tags?**  
A: Create tags, filter All/Untagged/custom. Batch tag in selection mode. Free: 5 tags. Premium: unlimited.

## Free vs Premium

**Free:** private gallery, import/export, 5 tags, ads (rewarded ad = 24h ad-free).  
**Premium (one-time):** no ads, unlimited tags, App Lock, cloud backup.

## Security & backup

**App Lock:** Premium. Settings → Security → PIN + biometric + auto-lock.  
**Cloud backup:** Premium. iCloud (iOS) or Google Drive (Android).  
**Encryption:** Coming soon in app Settings; media today is in private app directory.

## Export & delete

**Export:** per-item or batch → Export to Gallery (metadata preserved where supported).  
**Swipe delete:** swipe thumbnail → 5s undo snackbar.

## Account & data

**Delete account:** Settings → Account → Delete Account. See /hush-gallery/delete-account.  
**Uninstall:** local data removed unless exported; restore via cloud backup if enabled.
